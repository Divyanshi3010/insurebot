# InsureBot - AI Insurance Platform

## Quick Start

1. **Backend:**
   ```bash
   cd backend
   # Add your key to .env
   pip install -r requirements.txt
   python main.py
   ```

2. **Frontend:**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

## Features
- Gemini-powered Chat Interface
- RAG using CSV + JSON
- Accurate Term Insurance Logic
- Shadcn UI Response Design
